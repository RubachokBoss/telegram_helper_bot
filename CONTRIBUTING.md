# Руководство по внесению вклада

## Процесс разработки
1. Создайте issue для обсуждения новой функции
2. Создайте feature ветку от main
3. Напишите тесты для новой функциональности
4. Убедитесь, что все тесты проходят
5. Создайте Pull Request

## Стандарты кода
- Используйте gofmt для форматирования
- Пишите комментарии для публичных методов
- Следуйте принципам Clean Architecture
